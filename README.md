# 📊 Customer Shopping Trends SQL Analysis

> An end-to-end SQL data analysis project that demonstrates data cleaning, exploratory data analysis, business reporting, advanced SQL concepts, and Power BI dashboard development using a real-world customer shopping dataset.

---

# 📌 Project Overview

This project analyzes a **Customer Shopping Trends** dataset using **MySQL** and **Power BI**.Power BI project file is in process and will be uploaded soon. It follows a real-world data analyst workflow—from importing raw CSV data to generating actionable business insights.

The project showcases SQL skills ranging from basic queries to advanced analytical techniques, making it suitable for data analyst portfolio and interview preparation.

---

# 🎯 Project Objectives

- Import and manage retail data in MySQL
- Clean and validate raw customer data
- Perform exploratory data analysis (EDA)
- Solve real-world business problems using SQL
- Apply intermediate and advanced SQL concepts
- Build an interactive Power BI dashboard
- Generate business insights for decision-making
